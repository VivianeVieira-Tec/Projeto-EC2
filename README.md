# Projeto-EC2
Durante o projeto, realizei todo o processo de criação, configuração, monitoramento, segurança e escalabilidade de uma instância EC2, simulando atividades comuns executadas por profissionais de Cloud Computing e Infraestrutura.
Projeto AWS EC2 - Provisionamento e Gerenciamento de Instâncias
Sobre o Projeto

Este laboratório teve como objetivo colocar em prática conceitos fundamentais de Computação em Nuvem utilizando o Amazon EC2 (Elastic Compute Cloud), um dos principais serviços da AWS para provisionamento e gerenciamento de servidores virtuais.

Objetivos
Criar e configurar uma instância EC2
Configurar acesso de rede e regras de segurança
Disponibilizar uma aplicação web através do protocolo HTTP
Monitorar recursos da instância
Realizar escalabilidade vertical
Gerenciar armazenamento utilizando Amazon EBS
Implementar mecanismos de proteção da infraestrutura
Tecnologias Utilizadas
Amazon EC2
Amazon EBS
Security Groups
AWS Management Console
Redes e Segurança AWS
Atividades Realizadas

 Criação e configuração de uma instância EC2

 Configuração de rede e Security Groups

 Liberação de acesso HTTP para aplicação web

 Monitoramento e validação da instância

 Redimensionamento da instância de t3.micro para t3.small

 Expansão do volume EBS de 8 GiB para 10 GiB

 Teste de proteção contra encerramento da instância


Resultado Obtido

A instância EC2 foi provisionada com sucesso e disponibilizada para acesso através do endereço público fornecido pela AWS.

Durante o laboratório foi possível realizar alterações de capacidade computacional e armazenamento sem a necessidade de recriar o ambiente, demonstrando na prática os conceitos de flexibilidade, disponibilidade e escalabilidade da computação em nuvem.
